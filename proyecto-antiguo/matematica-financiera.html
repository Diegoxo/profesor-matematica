Matemática financiera

Temas frecuentes en matemática financiera.

  • Conversión de tasas de interés Aprenda como identificar una tasa nominal de una tasa efectiva y como aprender a convertir entre diferente tasa, como identificar que formula y cuando usarla.
  • Pago puntual Aprenda la formula de futuro de un pago puntual y como pasar a presente un pago futuro, aprenda a calcular el número de periodos cuando se tiene un pago presente y un pago futuro con una tasa dada.
  • Anualidad Aprenda todo lo relacionado con anualidades, anualidades vecindad y anticipadas, como hallar el valor presente y el valor futuro de una anualidad y como esta, esta afectada por la tasa de interés.
  • Gradientes aritméticos Aprenda como el gradiente aritmético se comporta de manera lineal, como usar la formula extensa para hallar los valora del primer pago y los valores sucesivos de un gradiente aritmético.
  • Gradiente geométrico Identificar el gradiente que va aumentando de manera geométrica con los periodos y como poder despejar la primera cuota de esta serie de pagos, descubra como iterar la tasa de interés en esta serie de pagos.
  • Gradientes aritméticos escalonado El gradiente aritmético escalonado, como lidiar con las diferentes tasas que este gradiente tiene y como utilizar las diferentes periodicidades de una misma tasa en un gradiente escalonado aritmético.
  • Gradiente geométrico escalonado Entienda como es el perfil geométrico por escalo para este tipo de pago, como esta incrementando en los diferentes escalones y como manejas con una misma tasa en diferentes periodicidades este gradiente.
  • Indexación de tasas Para entender que es una indexación, cuales son los tipos de indexación, cuáles son las indexaciones más usadas, como hacerlos cálculos de indexación, que formulas se usan para cada tipo de indexación.
  • Tablas de amortización Todo lo relacionado con tablas de amortización, los diferentes tipos de pagos, pago a cuota constante o pago a capital constante, dominar por completo y al dedillo. Entienda por completo como trabajar con tablas de amortización y como interpretarlas.
